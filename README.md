# ポイ活アプリ

日々の活動を記録してポイントを貯めるシンプルなアプリです。

## 公開ページ

**🌐 https://yokko405.github.io/point-activity-app/**

## 機能

- **ダッシュボード**: 月次カレンダーで日々の活動とポイントを視覚的に確認
- **記録**: 達成した活動を簡単に記録
- **アイテム管理**: カスタムアイテムの作成・編集（名前、ポイント、色、アイコン）
- **履歴**: 月ごとの累計ポイントを表示
- **目標設定**: 月次目標を設定してプログレスバーで進捗確認
- **ローカルストレージ**: データはブラウザに保存され、プライバシーが保護されます

## 使い方

1. `index.html` をブラウザで開く
2. 「アイテム」タブでポイント付き活動を追加
3. 「記録」タブで達成した活動を記録
4. 「ダッシュボード」でポイントと進捗を確認
